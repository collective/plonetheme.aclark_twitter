plonetheme.aclark_twitter Installation
--------------------------------------

To install plonetheme.aclark_twitter using zc.buildout and the plone.recipe.zope2instance
recipe, you can do this:

- Add ``plonetheme.aclark_twitter`` to the list of eggs to install, e.g.::

    [instance]
    eggs =
        plonetheme.aclark_twitter
       
- Re-run buildout, e.g. with::

    $ bin/buildout
